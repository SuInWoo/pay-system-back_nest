# 회의록 (Meetings)

팀 간 상의 결과를 정리한 회의록을 `docs/meetings/` 에 `.md` 파일로 보관합니다.

## 작성 규칙

- **파일명**: 주제를 알 수 있게 짧게 (예: `common-code-meeting.md`, `2025-03-주문API.md`).
- **구성**: 아래 템플릿을 참고해 **참석 팀**, **팀별 의견**, **결론**을 반드시 포함합니다.
- **저장 위치**: `docs/meetings/*.md` 만 사용합니다.

## 템플릿

`docs/meetings/템플릿.md` 를 복사해 새 회의록을 작성하세요.
